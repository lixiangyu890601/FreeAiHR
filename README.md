## 🔥 核心功能

### 智能筛选
- 📄 **AI简历评分**：自动评估简历完整度/质量
- 🔍 **人岗智能匹配**：NLP分析JD与简历匹配度
- 🎖 **候选人排名**：多维度的智能排序

### 招聘管理
- 🏷 **岗位发布**：多平台一键发布
- 📥 **简历收集**：统一人才库管理
