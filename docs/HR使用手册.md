## 1. 系统登录

### 1.1 访问系统

在浏览器中输入系统地址，进入登录页面。

### 1.2 登录操作

1. 输入用户名和密码
2. 点击「登录」按钮
3. 首次登录建议修改初始密码

> 💡 **提示**：如忘记密码，请联系系统管理员重置。

## 2. 首页概览

登录后进入系统首页，展示招聘核心数据：

### 2.1 数据统计卡片

| 指标 | 说明 |
|------|------|
| 在招职位数 | 当前正在招聘的职位总数 |

显示即将进行的面试安排，方便快速查看。

## 3. 职位管理

### 3.1 查看职位列表

路径：**招聘管理 → 职位管理**

职位列表显示所有职位信息，支持：
- 按职位名称、部门、状态筛选
- 查看职位详情
- 编辑/删除职位

### 3.2 发布新职位

1. 点击「新建职位」按钮
2. 填写职位信息：

| 字段 | 说明 | 必填 |
|------|------|------|
| 职位名称 | 如：Java高级工程师 | ✅ |
| 所属部门 | 选择招聘部门 | ✅ |
| 工作地点 | 工作城市 | ✅ |
| 招聘人数 | 计划招聘数量 | ✅ |
| 薪资范围 | 如：15K-25K | ✅ |
| 职位描述 | 工作职责说明 | ✅ |
| 任职要求 | 学历、经验、技能要求 | ✅ |
| 负责HR | 指定招聘负责人 | ✅ |

3. 点击「保存」完成发布

### 3.3 AI 生成职位描述

1. 在新建/编辑职位页面
2. 输入职位名称后，点击「AI生成」按钮
3. 系统自动生成职位描述和任职要求
4. 可根据需要修改生成内容

### 3.4 职位状态管理

| 状态 | 说明 |
|------|------|
| 招聘中 | 职位正在招聘，可接收简历 |
| 已暂停 | 暂停招聘，不再接收新简历 |
| 已关闭 | 招聘结束 |

### 3.5 分享职位

1. 在职位列表点击「分享」按钮
2. 复制公开招聘页面链接
3. 将链接发送给求职者或发布到招聘渠道

## 4. 简历管理

### 4.1 查看简历列表

路径：**招聘管理 → 简历管理**

### 4.2 上传简历

1. 点击「上传简历」按钮
2. 选择简历文件（支持 PDF、Word 格式）
3. 选择投递的目标职位（可选）
4. 点击「上传」

### 4.3 AI 智能解析

上传简历后，系统自动进行 AI 解析，提取：
- 基本信息（姓名、电话、邮箱）
- 教育背景（学校、专业、学历）
- 工作经历
- 技能标签

> 💡 **提示**：解析完成后建议核对信息准确性。

### 4.4 简历筛选

1. 使用筛选条件：
   - 学历要求
   - 工作年限
   - 期望薪资
   - 技能关键词

2. 查看简历详情
3. 标记简历状态：通过/不通过/待定

### 4.5 加入人才库

对于暂时不合适但有潜力的候选人：
1. 在简历详情页点击「加入人才库」
2. 候选人信息将保存到人才库，方便后续联系

## 5. 候选人管理

### 5.1 查看候选人

路径：**招聘管理 → 候选人管理**

显示所有进入招聘流程的候选人。

### 5.2 候选人状态流转

```
待筛选 → 筛选通过 → 面试中 → 面试通过 → 待入职 → 已入职
                ↓           ↓
              筛选淘汰     面试淘汰
```

### 5.3 更新候选人状态

1. 点击候选人进入详情页
2. 点击「更新状态」
3. 选择新状态并填写备注
4. 点击「确认」

### 5.4 查看候选人详情

候选人详情页包含：
- 基本信息
- 投递的职位
- 简历内容
- 面试记录
- 状态变更历史
- 评价记录

---

## 6. 面试管理

### 6.1 查看面试列表

路径：**招聘管理 → 面试管理**

### 6.2 安排面试

1. 在候选人详情页点击「安排面试」
2. 填写面试信息：

| 字段 | 说明 |
|------|------|
| 面试时间 | 选择日期和时间 |
| 面试官 | 选择面试官 |
| 面试类型 | 电话面试/视频面试/现场面试 |
| 面试轮次 | 一面/二面/三面/终面 |
| 面试地点 | 面试地址或会议链接 |
| 备注 | 其他说明 |

3. 点击「确认」完成安排

### 6.3 面试反馈

面试结束后：
1. 进入面试详情页
2. 点击「填写反馈」
3. 选择面试结果：通过/不通过/待定
4. 填写评价内容
5. 点击「提交」

### 6.4 面试日历

提供日历视图，直观展示面试安排。

## 7. 人才库

### 7.1 查看人才库

路径：**人才库**

人才库存储所有入库的候选人，便于后续招聘复用。

### 7.2 人才来源

- 简历解析后手动加入
- 往期招聘未录用但优质的候选人
- 主动收集的人才信息

### 7.3 推荐到职位

当有新职位开放时：
1. 在人才库中筛选合适人选
2. 点击「推荐」按钮
3. 选择目标职位
4. 候选人将自动进入该职位的招聘流程

### 7.4 人才搜索

支持多维度搜索：
- 姓名/手机号
- 技能关键词
- 学历
- 工作年限
- 期望薪资

## 10. 常见问题

### Q1: 简历解析不准确怎么办？

A: AI 解析可能存在误差，建议：
1. 上传格式规范的简历（PDF 最佳）
2. 解析后手动核对关键信息
3. 如有错误，可手动编辑修正

### Q2: 如何批量上传简历？

A: 目前支持单个上传，批量功能开发中。建议：
1. 一次选择多个文件上传
2. 或使用公开招聘页面让候选人自主投递

### Q3: 候选人状态如何修改？

A: 进入候选人详情页 → 点击「更新状态」→ 选择新状态。

### Q4: 如何让候选人自己投递简历？

A: 
1. 在职位列表点击「分享」
2. 复制公开招聘页面链接
3. 发送给候选人或发布到招聘平台

### Q5: AI 功能无法使用？

A: 请联系管理员检查：
1. AI 服务是否已配置
2. API 密钥是否有效
3. 账户余额是否充足

### Q6: 数据报表数据不准确？

A: 报表数据基于系统记录生成，请确保：
1. 所有简历都通过系统上传
2. 候选人状态及时更新
3. 选择正确的筛选条件
